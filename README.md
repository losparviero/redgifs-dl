# Redgifs-dl

Download gifs from Redgifs in the CLI.

For programmatic usage, see [redgif](https://npmjs.com/redgif).

### Usage

#### Without installing

```shell
npx redgifs-dl
```

Follow the prompts.

#### With install


```shell
npm i -g redgifs-dl
```

Then

```shell
redgifs-dl
```

Downloaded gif is stored in the current working directory.
